# 🚀 Data Structures Assignment – Stack & Queue (C)

## 📖 Overview

This repository contains C programs demonstrating the implementation and applications of **Stack** and **Queue** data structures.
Each program focuses on solving practical problems using basic operations like push, pop, enqueue, and dequeue.

## 📂 Programs List

### 🔁 Q1: String Reversal using Stack

Reverses a given string using stack operations.

#### ▶️ Execution

```
gcc Reverse_String.c
./a.out
```

#### 📌 Output

```
Enter string: STACK
Reversed string: KCATS
```

---

### ⚖️ Q2: Balanced Parentheses Checker

Checks whether an expression has properly matched parentheses.

#### ▶️ Execution

```
gcc Balanced_Parentheses.c
./a.out
```

#### 📌 Output

```
Input: (A+B)*(D+E)
Output: Balanced Expression

Input: ((A+B)
Output: Not Balanced
```

---

### 📈 Q3: Next Greater Element

Finds the next greater element for each element in an array.

#### ▶️ Execution

```
gcc Next_Greater_Element.c
./a.out
```

#### 📌 Output

```
Enter number of elements: 5
Enter elements: 6 3 9 2 5

Output:
6 -> 9
3 -> 9
9 -> -1
2 -> 5
5 -> -1
```

---

### 🖨️ Q4: Printer Queue Simulation

Simulates document printing using queue operations (FIFO).

#### ▶️ Execution

```
gcc Printer_Queue.c
./a.out
```

#### 📌 Output

```
1.Add 2.Print 3.Display 4.Exit
Enter document number: 101
Document 101 added to queue

Enter document number: 202
Document 202 added to queue

Pending documents: 101 202
Printing document 101
Pending documents: 202
```

---

### 🔄 Q5: Circular Queue Implementation

Implements a circular queue with menu-driven operations.

#### ▶️ Execution

```
gcc Circular_Queue.c
./a.out
```

#### 📌 Output

```
1.Enqueue 2.Dequeue 3.Peek 4.Display 5.Exit
Enter value: 5
Enter value: 15
Enter value: 25
Enter value: 35
Enter value: 50

Queue elements: 5 15 25 35 50
Front element: 5
Deleted: 5
Queue elements: 15 25 35 50
```
